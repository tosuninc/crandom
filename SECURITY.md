# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.5   | :white_check_mark: |
| < 1.0.5 | :x:                |

## Reporting a Vulnerability

1. Go to `Issues`.
2. Click `New`
3. Click `Report a vulnerability`
4. Please be descriptive as possible
